# Entropy Folding Cross-Domain Signal Atlas

This repository is a consolidation workspace for candidate cross-domain signals, correspondences, repeated motifs, and domain echoes already present in the source material.

It is intentionally cautious:

- It does not frame the material as proof.
- It does not add new signals by creative pattern-hunting.
- It preserves evidential ranking and uncertainty.
- It separates tool-derived outputs from direct textual source material.

## Important Framing Note

Many early atlas entries are internal/operator/tool-architecture traces from this project ecosystem. They are not presented as independent external validation. They are included because they document my own observations of where the proposed mechanism appears to pool, enact, or become operationally visible inside the development record.

The strongest entries are internal/operator/tool-architecture traces first; application essays are included later and should not be read as the primary evidential spine.

## Scope

This repo is for:

- mechanism traces
- supporting correspondences
- tool-derived candidate signals already present in existing outputs
- background and provenance material that contextualizes the atlas
- exclusions and tenuous links that should not be promoted into the main body

This repo is not for:

- hard-evidence claims
- theory extension
- new discovery runs
- including a source only because it is part of the broader Symbound ecosystem
